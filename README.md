## ✨ What is MarkusAI?

MarkusAI is a **CMO‑in‑a‑box**. Instead of managing a marketing team, you supervise
a roster of specialized AI agents (SEO, Writer, Reddit, Hacker News, X, LinkedIn,
GEO, Coding) from a single, opinionated dashboard.

The current release ships the full **operator UI**: dashboard, agents directory,
agent detail, content review workflow, ROI analytics, and a 3‑step onboarding flow.
The agents themselves are wired through a clean service layer so they can be plugged
to real LLMs / channels without touching the UI.

## 🧱 Architecture

The codebase follows a pragmatic **clean architecture**: the controllers stay thin,
domain data lives in immutable DTOs, and the predefined catalog data is encapsulated
in dedicated services that the controllers depend on. This keeps swapping the data
source for a real database or external API a one‑file change.

```diagram
                           ╭─────────────────────╮
                           │   React + Inertia   │  resources/js/pages/*
                           │   (presentation)    │
                           ╰──────────┬──────────╯
                                      │ props
                           ╭──────────┴──────────╮
                           │     Controllers     │  app/Http/Controllers/Marketing/*
                           │   (HTTP boundary)   │
                           ╰──────────┬──────────╯
                                      │ depends on
                           ╭──────────┴──────────╮
                           │      Services       │  app/Services/Marketing/*
                           │ (business / catalog)│
                           ╰──────────┬──────────╯
                                      │ returns
                           ╭──────────┴──────────╮
                           │        DTOs         │  app/Data/Marketing/*
                           │  (domain payloads)  │
                           ╰─────────────────────╯
```

### Backend (Laravel 13 / PHP 8.3)

| Layer          | Path                                  | Role                                     |
| -------------- | ------------------------------------- | ---------------------------------------- |
| DTOs           | `app/Data/Marketing/*`                | Immutable typed payloads                 |
| Services       | `app/Services/Marketing/*`            | In‑memory catalogs (Agent, Draft, etc.)  |
| Controllers    | `app/Http/Controllers/Marketing/*`    | Thin Inertia render layer                |
| Routes         | `routes/web.php`                      | Named, auth‑guarded routes               |

### Frontend (React 19 / Inertia 3 / Tailwind 4)

| Layer       | Path                                    | Role                                  |
| ----------- | --------------------------------------- | ------------------------------------- |
| Pages       | `resources/js/pages/{dashboard,agents,content,analytics,onboarding}` | Inertia routed views |
| Layout      | `resources/js/layouts/markus-layout.tsx`| Sidebar + topbar shell (dark + gold)  |
| Components  | `resources/js/components/markus/*`      | `MarkusCard`, `MarkusButton`, …       |
| Types       | `resources/js/types/marketing.ts`       | Shared TS shapes mirroring the DTOs   |
| Routes      | `resources/js/routes/*` (auto)          | Generated by Wayfinder                |

The marketing area enforces a fixed dark + gold (#D4AF37) palette — the rest of the
app keeps the existing light/dark toggle.

## 🚀 Getting started

### Prerequisites

- PHP **8.3+**
- Node.js **20+**
- Composer **2.x**
- SQLite (default) or any DB Laravel supports

### Install

```bash
git clone https://github.com/djascorp/markusai.git
cd markusai
composer install
npm install
cp .env.example .env
php artisan key:generate
php artisan migrate
```

### Run in development

```bash
composer run dev
```

This starts `php artisan serve`, the queue worker, and `vite` concurrently. Then
sign up at `http://localhost:8000`, verify your email, and you'll land on the CMO
dashboard.

### Build for production

```bash
npm run build
php artisan optimize
```

## 🗺️ Routes

| URL                    | Controller                                               | Page (Inertia)         |
| ---------------------- | -------------------------------------------------------- | ---------------------- |
| `/dashboard`           | `Marketing\DashboardController`                          | `dashboard`            |
| `/agents`              | `Marketing\AgentController@index`                        | `agents/index`         |
| `/agents/{agent}`      | `Marketing\AgentController@show`                         | `agents/show`          |
